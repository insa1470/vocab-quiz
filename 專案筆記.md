# 英文高頻單字複習程式 — 專案筆記

## 使用對象

| 孩子 | 年級 | 目標考試 | 單字範圍 |
|------|------|----------|----------|
| 女兒 | 高二 | 學測（明年） | 高中高頻字，約 2000–3000 字 |
| 兒子 | 國二 | 國中會考（明年） | 國中 2000 字（基礎 1200 + 進階 800） |

---

## 最終架構

```
原始 PDF（考卷 + 字表）
    ↓ scripts/01_download_pdfs.py
    ↓ scripts/02_analyze_pdfs.py（Claude API）
data/國中.json
data/高中.json
    ↓
web/（GitHub → Cloudflare → vocab.deepmystic.net）
    孩子手機瀏覽器複習
```

### 資料夾結構
```
/高中國中英文高頻單字/
    data/
        國中.json          ← 兒子的單字庫
        高中.json          ← 女兒的單字庫
    scripts/
        01_download_pdfs.py   ← 自動下載學測 PDF
        02_analyze_pdfs.py    ← Claude API 分析 PDF，生成 JSON
    web/
        index.html
        style.css
        app.js
        manifest.json      ← PWA（可安裝到手機桌面）
    原始資料/
        學測/              ← 110–114年 PDF（腳本自動下載）
        會考/              ← 手動放入
        字表/              ← 翰霖 PDF 等手動放入
    專案筆記.md
```

## 每筆單字資料格式

```json
{
  "word": "struggle",
  "phonetic": "/ˈstrʌɡəl/",
  "pos": "v.",
  "definition_zh": "掙扎；奮鬥",
  "level": "高中",
  "frequency_rank": 42,
  "example": "She struggled to finish the exam on time.",
  "example_zh": "她努力在時間內完成考試。",
  "usage": [
    { "pattern": "struggle to V", "meaning_zh": "努力嘗試做某事" },
    { "pattern": "struggle with sth", "meaning_zh": "與某事搏鬥" }
  ]
}
```

## Web App 功能

- **選擇畫面**：進入選「國中」或「高中」
- **答題流程**：
  1. 看例句（空格），選出正確單字（4選1）
  2. 翻面：顯示音標、中文、完整例句、用法（可展開）
  3. 按「認識」或「再複習」
- **進度追蹤**：「再複習」的字存在 localStorage，下次優先出現
- **PWA**：可安裝到手機桌面，支援離線

## 資料來源

### 學測試題（自動下載）
- 升學王 go100.com.tw，110–114年 PDF，腳本自動下載
- URL 規律：`go100.com.tw/file/exam/S{年份}-GSAT/{年份}GSAT_English.pdf`

### 會考試題（手動下載）
- 心測中心：cap.rcpet.edu.tw/examination.html
- 下載近 5 年放入 `原始資料/會考/`

### 字表（手動下載）
- 翰霖 國中基礎 1200：han-lin.tw/english-all-words/
- 翰霖 國中進階 800：han-lin.tw/advanced-800-words/
- Teacherfrog 學測頻率表：teacherfrog-class.blogspot.com/2019/01/93-107Voc.html
